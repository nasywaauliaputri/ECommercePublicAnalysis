# E-Commerce Public Dataset Dashboard
Dashboard ini dibuat untuk menganalisis performa bisnis e-commerce berdasarkan data transaksi yang telah melalui proses data cleaning.

---

## Setup Environment

### 🔹 Anaconda
```bash
conda create --name ecom-ds python=3.9
conda activate ecom-ds
pip install -r requirements.txt
```
### 🔹 Terminal / Shell
```bash
mkdir ecommerce-dashboard
cd ecommerce-dashboard
pip install -r requirements.txt
```

### ▶️ Menjalankan Dashboard
```bash
streamlit run dashboard.py
```

### 📁 Dataset
Gunakan dataset yang sudah dibersihkan:
```bash
cleaned_merged_data.csv
```
